# 🛡️ Quests - TodoList App

Una aplicación web dinámica desarrollada con Flask para la gestión de tareas (Quests).
Este proyecto permite a los usuarios registrarse, iniciar sesión de forma segura
y administrar sus lista de objetivos diarios a través de una interfaz limpia y estructurada.

## 🛠️ Tecnologías Utilizadas

- Backend:Python, Flask
- Base de Datos: SQLite (gestionado con SQLAlchemy/módulo interno)
- Frontend: HTML5, CSS3, Jinja2
- Entorno: Entorno virtual nativo de Python (`venv`)
